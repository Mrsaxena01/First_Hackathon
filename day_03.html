      Day - 03 : HTML - Nested Lists & Tables

In this lesson we will study two important HTML structures: Nested Lists and Tables. These are used to organize and represent data clearly on a webpage.


  1. Nested Lists (<ul>, <ol>, <li>)

    2. What it is: Nested lists mean lists inside another list. We can nest unordered lists inside ordered lists and vice-versa.

    Tags used:

    • <ul> : Creates an Unordered (bulleted) List
    • <ol> : Creates an Ordered (numbered) List
    • <li> : List item element, used inside ul/ol

    Purpose: Helps in representing hierarchical data or breaking topics into sub-topics clearly.



  2. HTML Tables

    3. What it is: Tables allow us to represent data in rows and columns format.

    Tags used:

    • <table> : Defines the table
    • <tr> : Table Row
    • <td> : Table Data (cell)
    • <th> : Table Header (bold & centered by default)
    • <thead> : Groups the header section of the table
    • <tbody> : Groups the main body content of the table
    • <tfoot> : Groups the footer content of the table
    • <caption> : Adds a table title

    Purpose: Tables help organize data like marksheets, employee data, schedules, etc., into a grid for easy reading.



  3. Attributes in Table

    • border : To show borders around table cells
    • cellspacing : Space between cells
    • cellpadding : Space inside a cell (between text and border)
    • colspan : Merge multiple columns
    • rowspan : Merge multiple rows
